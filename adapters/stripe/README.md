# @godmode-cli/stripe

Stripe adapter for [Godmode](https://godmode.so).

## Install

```sh
godmode add stripe
```

## Environment

```sh
export STRIPE_API_KEY=sk_...
```

## Claude Code

```json
{
  "mcpServers": {
    "stripe": {
      "command": "godmode",
      "args": ["mcp", "stripe"]
    }
  }
}
```
