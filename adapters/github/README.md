# @godmode-cli/github

GitHub adapter for [Godmode](https://godmode.so).

## Install

```sh
godmode add github
```

## Environment

```sh
export GITHUB_TOKEN=ghp_...
```

## Claude Code

```json
{
  "mcpServers": {
    "github": {
      "command": "godmode",
      "args": ["mcp", "github"]
    }
  }
}
```
