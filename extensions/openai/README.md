# @godmode-cli/openai

OpenAI extension for [Godmode](https://godmode.so).

## Install

```sh
godmode ext install openai
```

## Environment

```sh
export OPENAI_API_KEY=sk-...
```

## Claude Code

```json
{
  "mcpServers": {
    "openai": {
      "command": "godmode",
      "args": ["openai", "mcp"]
    }
  }
}
```
