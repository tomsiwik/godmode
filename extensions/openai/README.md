# @godmode-cli/openai

OpenAI extension for [Godmode](https://godmode.so).

## Install

```sh
godmode extension add openai
```

## Environment

```sh
export OPENAI_API_KEY=sk-...
```

## Claude Code

```json
{
  "mcpServers": {
    "openai": {
      "command": "godmode",
      "args": ["mcp", "openai"]
    }
  }
}
```
