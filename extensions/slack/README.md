# @godmode-cli/slack

Slack extension for [Godmode](https://godmode.so).

## Install

```sh
godmode ext install slack
```

## Environment

```sh
export SLACK_TOKEN=xoxb-...
```

## Claude Code

```json
{
  "mcpServers": {
    "slack": {
      "command": "godmode",
      "args": ["slack", "mcp"]
    }
  }
}
```
