# @godmode-cli/context7

Context7 extension for [Godmode](https://godmode.so).

## Install

```sh
godmode ext install context7
```

## Environment

```sh
export CONTEXT7_API_KEY=...
```

## Claude Code

```json
{
  "mcpServers": {
    "context7": {
      "command": "godmode",
      "args": ["context7", "mcp"]
    }
  }
}
```
