# @godmode-cli/claude-code-channels

Inter-session messaging for Claude Code instances via [Godmode](https://godmode.so).

## Install

```sh
godmode ext install claude-code-channels
```

## Claude Code

```json
{
  "mcpServers": {
    "channels": {
      "command": "godmode",
      "args": ["claude-code-channels", "mcp"]
    }
  }
}
```
