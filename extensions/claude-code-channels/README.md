# @godmode-cli/claude-code-channels

Inter-session messaging for Claude Code instances via [Godmode](https://godmode.so).

## Install

```sh
godmode extension add claude-code-channels
```

## Claude Code

```json
{
  "mcpServers": {
    "channels": {
      "command": "godmode",
      "args": ["mcp", "claude-code-channels"]
    }
  }
}
```
