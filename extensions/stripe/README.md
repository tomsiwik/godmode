# @godmode-cli/stripe

Stripe extension for [Godmode](https://godmode.so).

## Install

```sh
godmode ext install stripe
```

## Environment

```sh
export STRIPE_API_KEY=sk_...
```

## Claude Code

```json
{
  "mcpServers": {
    "stripe": {
      "command": "godmode",
      "args": ["stripe", "mcp"]
    }
  }
}
```
