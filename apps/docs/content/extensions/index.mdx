## Overview

Godmode ships with extensions for popular APIs. Each extension is an npm package under `@godmode-cli/*`.

```bash
godmode extension add stripe
godmode extension add github
```

## Available Extensions

| Extension | Type |
|---------|------|
| [Stripe](/extensions/stripe) | REST |
| [OpenAI](/extensions/openai) | REST |
| [Slack](/extensions/slack) | REST |
| [GitHub](/extensions/github) | GraphQL |
| [Context7](/extensions/context7) | MCP |
| [Claude Code Channels](/extensions/claude-code-channels) | MCP |
