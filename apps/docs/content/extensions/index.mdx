---
title: Extensions
description: Installable extensions grouped by interface type.
---

Install any of these with `godmode ext install <slug>`, then invoke with `godmode <slug> <interface> ...`. Each page covers auth setup, live invocation examples, and the MCP config snippet.

| Extension | Interface | Slug |
|-----------|-----------|------|
| [Stripe](./stripe) | REST | `stripe` |
| [OpenAI](./openai) | REST | `openai` |
| [Slack](./slack) | REST | `slack` |
| [Petstore](./petstore) | REST | `petstore` |
| [GitHub](./github) | GraphQL | `github` |
| [Context7](./context7) | MCP | `context7` |
| [Claude Code Channels](./claude-code-channels) | MCP | `claude-code-channels` |

To contribute your own, see [Submit an extension](./submit) and the [Developers → Authoring](/developers) section.
