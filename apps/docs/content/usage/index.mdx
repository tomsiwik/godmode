## Install

```bash
npm install -g godmode
```

## Add an extension

```bash
godmode extension add stripe
```

## Use it

```bash
godmode api stripe customers cus_123
godmode mcp stripe
```

## Claude Code

```json
{
  "mcpServers": {
    "stripe": {
      "command": "godmode",
      "args": ["mcp", "stripe"]
    }
  }
}
```
