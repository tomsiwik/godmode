## REST — Stripe

```yaml
slug: stripe
name: Stripe
type: api
spec: https://raw.githubusercontent.com/stripe/openapi/.../spec3.yaml
url: https://api.stripe.com
auth:
  env: STRIPE_API_KEY
  type: bearer
```

## GraphQL

```yaml
slug: my-gql
name: My GraphQL API
type: graphql
url: https://api.example.com/graphql
auth:
  env: MY_API_KEY
```

## MCP — Context7

```yaml
slug: context7
name: Context7
type: mcp
url: https://mcp.context7.com/mcp
auth:
  env: CONTEXT7_API_KEY
  type: api-key
```

## Smithery

Any MCP server on Smithery works out of the box:

```yaml
slug: instagram
name: Instagram
type: mcp
url: https://server.smithery.ai/instagram/mcp
auth:
  env: SMITHERY_API_KEY
```
