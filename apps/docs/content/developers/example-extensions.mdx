---
title: Petstore
description: A minimal REST extension with no auth
---

## Manifest

```yaml
slug: petstore
name: Petstore
type: api
spec: https://petstore3.swagger.io/api/v3/openapi.json
url: https://petstore3.swagger.io/api/v3
```

## Usage

```bash
godmode extension add petstore
godmode api petstore pet 1
godmode mcp petstore
```
