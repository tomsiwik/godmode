## Manifest

Create a `manifest.yaml` in any folder:

```yaml
slug: my-api
name: My API
type: api
spec: https://example.com/openapi.json
url: https://api.example.com
auth:
  env: MY_API_KEY
  type: bearer
headers:
  X-Custom: value
```

## Register

```bash
godmode add ./my-adapter
godmode my-api --help
godmode mcp my-api
```

## GraphQL

```yaml
slug: my-gql
name: My GraphQL API
type: graphql
spec: https://example.com/schema.graphql
url: https://api.example.com/graphql
auth:
  env: MY_API_KEY
```

## MCP

```yaml
slug: my-server
name: My MCP Server
type: mcp
url: https://my-mcp-server.com/mcp
auth:
  env: MY_API_KEY
```

## Testing

```ts
import { testAdapter } from 'godmode/test/adapter';
testAdapter('my-api', 'path/to/manifest.yaml');
```
