title: Developers
description: Build, publish, and customize godmode extensions.
---

Everything you need to ship an extension — from the manifest schema to publishing on npm.

- **Authoring** — write and test a manifest locally, then publish as `@godmode-cli/<name>`.
- **Reference** — manifest schema, interface types, auth types.
- **Advanced** — custom help classes, MCP derivative serving, multi-interface extensions.
- **Contributing** — work on the CLI itself.
