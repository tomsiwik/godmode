## Overview

Godmode ships with adapters for popular APIs. Each adapter is an npm package under `@godmode-cli/*`.

```bash
godmode add stripe
godmode add github
```

## Available Adapters

| Adapter | Type |
|---------|------|
| [Stripe](/adapters/stripe) | REST |
| [OpenAI](/adapters/openai) | REST |
| [Slack](/adapters/slack) | REST |
| [GitHub](/adapters/github) | GraphQL |
| [Context7](/adapters/context7) | MCP |
| [Claude Code Channels](/adapters/claude-code-channels) | MCP |
