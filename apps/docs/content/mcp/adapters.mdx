## Any Smithery Server

Any MCP server hosted on Smithery works:

```yaml
name: Instagram
type: mcp
url: https://server.smithery.ai/instagram/mcp
auth:
  env: SMITHERY_API_KEY
```

## Creating an Adapter

```yaml title="apis/myserver/myserver.yaml"
name: MyServer
type: mcp
url: https://my-mcp-server.com/mcp
auth:
  env: MY_API_KEY
```

```ts title="apis/myserver/myserver.test.ts"
import { testAdapter } from '../../test/adapter';
testAdapter('myserver', 'apis/myserver/myserver.yaml');
```
