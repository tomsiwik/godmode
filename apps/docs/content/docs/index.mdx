## Install

```bash
npm install -g godmode
```

## Quick Start

```bash
godmode add stripe
godmode stripe customers cus_123
godmode mcp stripe
```

## Claude Code

```json
{
  "mcpServers": {
    "stripe": {
      "command": "godmode",
      "args": ["mcp", "stripe"]
    }
  }
}
```

## Supported Protocols

| Type | Config | Discovery |
|------|--------|-----------|
| REST / OpenAPI | `type: api` | Parses OpenAPI spec |
| GraphQL | `type: graphql` | Introspection or SDL file |
| MCP | `type: mcp` | `tools/list` via JSON-RPC |
