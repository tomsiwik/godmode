## Creating an Adapter

```yaml title="apis/myapi/myapi.yaml"
name: MyAPI
type: api
spec: https://example.com/openapi.json
url: https://api.example.com
auth:
  env: MYAPI_TOKEN
```

```ts title="apis/myapi/myapi.test.ts"
import { testAdapter } from '../../test/adapter';
testAdapter('myapi', 'apis/myapi/myapi.yaml');
```
